NeuralNetwork::NeuralNetwork ( NeuralNetwork &  p_copy,
bool  p_clone = false 
)

Creates a copy or clone of the network.

The copy shares the parameters with the original and every change will have impact on the original as well as on the copy. The clone has its own parameters with the same value as the original and change in parameters of one network will leave the others network parameters intact.

Parameters
p_copysource network
p_clonecopy / clone flag
